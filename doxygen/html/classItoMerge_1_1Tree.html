Class for making a tree structure with ItoMerge::Node<T> in it.

This is the main function that is used for partitioning particles ala ItoSolver – it uses the ItoMerge::Node<T> structure to recursively partition a list of primitives into a new list of primitives.

Note
This class should work with any Node<T> – refer to ItoMerge::Node<T> for requirements on the template parameters. To build the tree the user will just call buildTree() which selects the first splitting plane and number of leaves in the tree. The user can also input his own partitioning function which determines how leaf nodes are split.
