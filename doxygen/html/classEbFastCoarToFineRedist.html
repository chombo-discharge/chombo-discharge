Detailed Description

EBFineToCoarRedist with faster define function.

This is just like Chombo's EBCoarToFineRedist except that it is much faster to define.

Note
To use run-time debugging and profiling, use EbFastCoarToFineRedist.debug=true and EbFastCoarToFineRedist.profile=true
