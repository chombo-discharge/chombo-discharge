Detailed Description

EBFineToCoarRedist with faster define function.

This is just like Chombo's EBCoarToCoarRedist except that it is much faster to define.

Note
To use run-time debugging and profiling, use EbFastCoarToCoarRedist.debug=true and EbFastCoarToCoarRedist.profile=true
