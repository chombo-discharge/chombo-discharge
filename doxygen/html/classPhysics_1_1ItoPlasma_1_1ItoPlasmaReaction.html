Detailed Description

Reaction class for ItoPlasmaPhysics.

To use this class you should initialize a reaction e.g. ItoPlasmaReaction({0,1,2}, {0,4}, {0}) which is equivalent to the reaction X_0 + X_1 + X_2 => X_1 + X_4 + Y_0.
