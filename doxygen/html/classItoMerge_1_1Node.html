template<class T>
class ItoMerge::Node< T >

Class for describing a single node with two children and one parent. This is used together with ItoMerge::Tree for particle handling.

template<class T >
void ItoMerge::Node< T >::split ( const int  a_dir)
inline

Split node along the input coordinate.

This will split the node along the input coordinate and make this node into a regular node. The left/right nodes become new leaf nodes.

Parameters
[in]a_dirSplitting coordinate – this will compute a coordinate plane and put about half the mass on one side and half the mass on the other.
