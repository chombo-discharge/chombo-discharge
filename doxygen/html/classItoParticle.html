Detailed Description

A particle class for use with ItoSolvers, i.e. drifting Brownian walkers.

This class is used to encapsulate the requirements for running an ItoSolver. This computational particle contains position, mass, velocity, as well as a diffusion coefficient, a mobility, energy, and previous particle position. It also contains a temporary, and users can choose to allocate additional memory at runtime. This memory is added on top of the other particle fields. In total, this class stores 5 + 3*SpaceDim fields. So in 3D this particle class contains 14 Reals => 112 bytes.
