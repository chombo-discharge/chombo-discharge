5408  // position A to position B, excluding friction from collision with other molecules. We compute this energy increase as
5409  //
5410  // q * V(B) - V(A)
5411  //
5412  // which means that the energy rate is q*(V(B) - V(A))/a_dt.

5416  // over into the particle realm and we interpolate V(B) and V(A) onto some storage in the particle container. We then
5417  // assign an effective weight w * [V(B) - V(A)] to the particles which we deposit onto the mesh using the appropriate
5418  // deposition scheme that the user has assgned.
