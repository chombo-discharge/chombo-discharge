Detailed Description

Version of EBFluxRegister with faster define function.

This is just like EBFluxRegister except it has a much faster define function. This class exists solely because the original Chombo definition has crazy nested LayoutIterators, which break when are many boxes.

Note
To use run-time and profiling, use EbFastFluxRegister.profile=true
