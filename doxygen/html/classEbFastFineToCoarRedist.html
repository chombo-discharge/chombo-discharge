Detailed Description

EBFineToCoarRedist with faster define function.

This is just like Chombo's EBFineToCoarRedist except that it is much faster to define.

Note
To use run-time debugging and profiling, use EbFastFineToCoarRedist.debug=true and EbFastFineToCoarRedist.profile=true
