Detailed Description

Namespace for various merging utilities for ItoSolver.

This namespace is just a collection of various utils. In general we tend to use the ItoMerge::Tree for making a hierarchical view of the particles in a particular grid cell. The user can use that tree structure together with the default (or specified) partitioning function for partitioning the particles into a tree structure. Currently, this tree structure is a kd-tree.
